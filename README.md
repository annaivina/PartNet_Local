# PartNet_Local
ParticleNet and Partcile transformer code, which is adapted from https://github.com/jet-universe/particle_transformer/ 

This code uses two models (PartcleNet and Transformers) to train events.

Currently, this code is modified to use our new data, which has been processed by cocoa detector 

The reprocessing of the data is in /reprocessing directory

The original code will be used to train our customer-made data, which includes the GNN jet coming from the HGPflow GNN package.

The data visualization is summarized in the code from plots/ folder.
